## 📖 About The Project

This portfolio website is designed to serve as a central hub for my work, achievements, and technical skills. It reflects my passion for web development and clean UI/UX design.

The goal of this project is to:

* Present my projects in a structured and visually appealing way
* Highlight my technical expertise
* Provide an easy way for recruiters/clients to contact me

### 👤 Personal Introduction

* Brief bio and career overview
